# Security Policy

Author: Alexandre VANDEMOORTELE (@GoXLd)

## Supported Versions

This project is maintained on the `main` branch.
Security fixes are applied to the latest code in `main`.

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories:

- Repository -> Security -> Advisories -> Report a vulnerability

If that is not available, open a private contact request to the maintainer:

- GitHub: https://github.com/GoXLd

Please include:

- vulnerability description
- reproduction steps
- impact assessment
- optional fix proposal

## Disclosure

Please do not publish details of a vulnerability before a fix is available.
A coordinated disclosure process helps protect users of the project.
